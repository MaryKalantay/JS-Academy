Реализовать тултипы

Тултипы должны работать на все <a> у которых есть аттрибут title

Реализовать можно так:

Создать <div id="tooltip"> - на jQuery это очень просто: var tooltip = $('<div id="tooltip">');
Вставить созданный <div> в <body> - для этого можно использовать код: tooltip.appendTo('body');
Перебираем все ссылки, у которых есть title. Записываем title в какую-то переменную, а у ссылки удаляем аттрибут title (чтобы при ховере не появлялся дефолтный).
Добавляем обработчики событий:
При mouseenter - в tooltip вставить текст из переменной title. отобразить tooltip
При mousemove - спозиционировать блок. Координаты мыши можно узнать относительно <body> используя информацию события: btn.mousemove(function(e){ e.pageX и e.pageY})
При mouseleave - спрятать блок тултипа
#tooltip c текстом можно относительно сдвинуть с помощью margin'a (используя метод jQuery .css его нужно немного отодвинуть от курсора мыши)
