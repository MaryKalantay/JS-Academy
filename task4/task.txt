Необходимо потянуть стоящие рядом блоки по максимальной высоте.


Написать три функции, которые потянут по одинаковой высоте:

Два блока <div id="block1"> и <div id="block2">
Дочерние дивы внутри <div id="list-holder">
Три дива с классами element, container, my-box (обратите внимание что у них есть padding)
Приблизительный алгоритм решения задачи:

Нужно реализовать несколько функций - по одной для решения конкретного пункта задания. 
Находим блоки которые нужно сравнить, перебираем их с помощью цикла, находим максимальную высоту. 
После этого снова перебираем блоки и устанавливаем им высоту равную значению максимальной.
