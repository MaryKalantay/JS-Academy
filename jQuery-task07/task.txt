Кастомные формы (на примере <input type="checkbox">Expand/Collapse

Реализовать кастомный checkbox. 
Пройтись по всем checkbox и создать рядом с каждым элемент <span class="checkbox-area"> используя метод insertBefore. 
Созданный элемент должен соответствовать значению checkbox (когда checkbox отмечен 
будет дополнительно добавляться класс checkbox-area-checked, иначе - удаляться). Также при клике по фейковому 
чекбоксу должен переключаться и реальный.
